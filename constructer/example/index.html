Powerful Features

Everything you need to create stunning visual experiences

Path Builder

Fluent API for creating complex SVG paths at runtime. Stars, polygons, curves - all programmatic.

Path Morphing

Smooth transitions between any two shapes. No matter how different the paths are.

Gradient Presets

Beautiful gradient mixins - sunset, ocean, aurora, neon, mesh gradients and more.

SVG Filters

Blur, glow, drop shadows, color matrices - all generated and applied at runtime.

Timeline Animation

Chain and stagger animations with a simple API. Full control over timing and easing.

Scroll Effects

Scroll-linked animations, parallax, and intersection observers built in.

@use 'constructer' as c;

// Gradient backgrounds
.hero {
  @include c.c-gradient-mesh-neon();
}

// Glass morphism
.card {
  @include c.c-glass(12px, 0.1);
}

// Typography effects
.title {
  @include c.c-text-gradient-aurora();
  @include c.c-fluid-text(2rem, 4rem);
}

// Textures
.overlay {
  @include c.c-grain(0.05);
  @include c.c-scanlines(0.1);
}

// Responsive
@include c.c-above('md') {
  .grid { columns: 3; }
}
import Constructer from 'constructer';

const { path, svg, filters, morph, timeline } = Constructer;

// Create SVG shapes
const star = path()
  .star(100, 100, 50, 25, 5)
  .build({ fill: '#667eea' });

// Apply filters
const { id, filter } = filters.glow(8, '#00f2fe');
filters.addFilterToSVG(mySvg, filter);
filters.applyFilter(star, id);

// Morph between shapes
await morph.morph(element, starPath, circlePath, {
  duration: 600,
  easing: 'ease-in-out'
});

// Timeline animations
timeline()
  .stagger(elements, keyframes, 400, 100)
  .play();
